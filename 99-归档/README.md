# 99-归档

> 存放历史归档文件，已完成的项目/内容归档。

## 目录结构

```
99-归档/
├── [年份]/
│   ├── Q1/             # 一季度归档
│   ├── Q2/             # 二季度归档
│   ├── Q3/             # 三季度归档
│   └── Q4/             # 四季度归档
└── README.md
```

## 使用说明

1. 按年份+季度归档
2. 归档内容只读不修改
3. 保持原始目录结构

## 命名规范

```
[年份]-[季度]-[原名称]
示例：2026-Q1-知识沉淀-V1.zip
```
