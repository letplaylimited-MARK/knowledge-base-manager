# 04-输出成果

> 存放项目产出物：方案、报告、代码、内容等。

## 目录结构

```
04-输出成果/
├── 方案/            # 方案文档
├── 报告/            # 分析报告
├── 代码/            # 代码文件
├── 内容/            # 文本内容
└── README.md
```

## 使用说明

1. 方案：存放设计文档、规划文档
2. 报告：存放数据分析报告、总结报告
3. 代码：存放生成的代码文件
4. 内容：存放生成的文本内容

## 命名规范

```
[项目]-[类型]-[日期].[扩展名]
示例：AI话术-需求文档-20260316.docx
```
