# 03-资产库

> 存放可复用的资产：提示词、模板、脚本、工具等。

## 目录结构

```
03-资产库/
├── 提示词/          # Prompt模板
├── 模板/            # 文档模板
├── 脚本/            # 自动化脚本
├── 工具/            # 工具配置
└── README.md
```

## 使用说明

1. 提示词：存放验证过的Prompt模板
2. 模板：存放文档模板
3. 脚本：存放自动化脚本
4. 工具：存放工具配置

## 命名规范

```
[类型]-[名称]-[版本].[扩展名]
示例：Prompt-风控审核-终极版.md
```
