# 知识库框架 V2.1 产品规划

## 版本目标
在 V2.0 正式交付的基础上，V2.1 聚焦于「开箱即用体验提升」和「数据管道可靠性」。

## 需求列表

### P0 (必须)
1. **端到端演示数据包** — 交付包自带5+个演示文件 + 预构建索引，用户 clone 后可直接搜索
2. **一键初始化脚本** — `python setup.py` 完成依赖安装→索引构建→启动服务全流程
3. **默认配置文件** — `config/` 目录不再为空，提供开箱即用的配置

### P1 (应该)
4. **搜索性能优化** — 查询延迟 <500ms (当前~320ms)
5. **批量导入模式** — 支持目录级导入，有进度条和错误统计（原型已完成）
6. **增量索引更新** — 避免每次全量重建，仅更新变更文件

### P2 (可以)
7. **Web UI 实时预览** — 搜索页面支持Markdown渲染预览
8. **多语言搜索** — 支持中英文混合查询（当前中文为主）
9. **记忆可视化** — 在Web UI中以时间线/网络图形式展示记忆系统

## 技术路线

```
V2.0 (当前) ──→ V2.1 演示数据+初始化 ──→ V2.2 性能优化 ──→ V3.0 多模态支持
```
