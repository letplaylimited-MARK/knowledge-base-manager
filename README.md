# 通用知识库框架 V2.0

> 版本：V2.0 | 更新：2026-05-17

---

## 简介

通用知识库框架是一个面向**任何行业/项目**的知识管理解决方案。支持：

- 知识图谱管理
- AI协作系统
- 长期记忆系统
- 团队协作
- 自动化工作流

## 快速开始

1. 运行 `00-快速开始/setup.bat` 初始化
2. 查看 `00-快速开始/README.md`
3. 开始添加你的知识

## 目录结构

```
├── 00-快速开始/        ← 入门引导
├── 01-收件箱/          ← 新知识入口
├── 02-对话记录/        ← 对话存档
├── 03-资产库/          ← 可复用资产
├── 04-输出成果/        ← 产出物
├── 05-知识沉淀/        ← 知识库
├── 06-参考资料/        ← 参考资料
├── 07-项目文档/        ← 项目文档
├── 99-归档/            ← 历史归档
└── .workbuddy/         ← 系统配置
```

## 核心功能

| 功能 | 说明 |
|------|------|
| 知识图谱 | 05-知识沉淀/wiki/ 目录结构 |
| 长期记忆 | .workbuddy/记忆层/ |
| AI协作 | .workbuddy/AI协作体系/ |
| 自动化脚本 | 18个Python脚本 |
