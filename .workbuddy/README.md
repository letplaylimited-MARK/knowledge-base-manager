## 🚀 使用指南

### 1. 新任务开始

```
1. 读 记忆层/MEMORY.md → 了解背景
2. 查 流程/智能分流引擎.md → 判断类型
3. 选 合适流程 → 启动执行
4. 用 流程/任务验证标准.md → 确保质量
5. 填 流程/交接文档模板.md → 完整归档
```

### 2. 新想法/构思

```
1. 读 流程/思绪整理系统.md → 了解流程
2. 直接对话 → AI帮你整理
3. 形成结晶 → 归档到05-知识沉淀
4. 更新 记忆层/MEMORY.md → 同步记忆
```

### 3. 每日开始

```
1. 检查 01-收件箱/ → 有新任务？
2. 回顾 记忆层/MEMORY.md → 了解上下文
3. 开始工作 → 使用七角色模式
4. 结束时 → 验证+交接+记忆更新
```
