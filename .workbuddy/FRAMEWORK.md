## 六、自动化脚本

### 6.1 必装脚本

```python
# 工作台根目录创建 scripts.bat

@echo off
echo 工作台自动化脚本
echo 1. 更新索引 (python .workbuddy/scripts/update_index.py)
echo 2. 搜索内容 (python .workbuddy/scripts/search_content.py)
echo 3. 备份数据 (python .workbuddy/脚本/backup.py)
echo 4. 收件箱监控 (python .workbuddy/scripts/enhanced_inbox_watcher.py)

set /p choice=请选择:
if "%choice%"=="1" python .workbuddy/scripts/update_index.py
if "%choice%"=="2" set /p kw=请输入关键词: && python .workbuddy/scripts/search_content.py "%kw%"
if "%choice%"=="3" python .workbuddy/脚本/backup.py
if "%choice%"=="4" python .workbuddy/scripts/enhanced_inbox_watcher.py
```
