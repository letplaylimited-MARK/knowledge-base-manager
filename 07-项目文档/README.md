# 07-项目文档

> 存放项目特有文档：需求文档、设计文档、执行记录、复盘总结。

## 目录结构

```
07-项目文档/
├── 需求文档/        # PRD/需求
├── 设计文档/        # 设计/方案
├── 执行记录/        # 执行日志
├── 复盘总结/       # 复盘文档
└── README.md
```

## 使用说明

1. 需求文档：存放需求分析、功能规格
2. 设计文档：存放技术设计、架构设计
3. 执行记录：存放执行日志、周报
4. 复盘总结：存放复盘文档、经验总结

## 命名规范

```
[项目]-[类型]-[版本]-[日期].[扩展名]
示例：知识库-需求文档-V1.0-20260421.md
```
