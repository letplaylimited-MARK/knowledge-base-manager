## 5分钟入门

### 第1步：运行初始化脚本

```bash
cd 通用知识库框架
00-快速开始\setup.bat
```

### 第2步：了解目录结构

| 目录 | 用途 |
|------|------|
| 00-快速开始/ | 入门引导 |
| 01-收件箱/ | 新知识入口 |
| 05-知识沉淀/ | 知识库核心 |
| .workbuddy/ | 系统配置 |

### 第3步：添加第一个知识

1. 在 `05-知识沉淀/wiki/concepts/` 创建新概念
2. 在 `05-知识沉淀/wiki/entities/` 创建新实体
3. 运行 `python .workbuddy/scripts/update_index.py` 更新索引

### 第4步：使用搜索

```bash
python .workbuddy/scripts/search_content.py "关键词"
```

## 核心操作

### 添加知识

```
1. 创建文件 → 添加内容 → frontmatter
2. 更新 wiki/index.md → 添加链接
3. 运行 update_index.py
```

### 使用AI协作

```
在 AGENTS.md 中配置AI参数
```

### 自动化任务

```bash
# 更新索引
python .workbuddy/scripts/update_index.py

# 搜索内容
python .workbuddy/scripts/search_content.py "关键词"
```

## 常见问题

### Q: 如何修改AI模型配置？
A: 编辑 `.workbuddy/AI协作体系/模型配置/` 目录下的YAML文件

### Q: 如何添加新知识？
A: 将文件放入 `01-收件箱/` → 运行 update_index.py

### Q: 如何备份？
A: 运行 `python .workbuddy/scripts/backup.py`
