# Universal Knowledge Base Framework V2.0

> Version: V2.0 | Updated: 2026-05-23

---

## Introduction

A universal knowledge management framework for **any industry/project**. Supports:

- Knowledge Graph Management
- AI Collaboration System
- Long-term Memory System
- Team Collaboration
- Automated Workflows

## Quick Start

1. Run `00-快速开始/setup.bat` to initialize
2. Check `00-快速开始/README.md`
3. Start adding your knowledge

## Directory Structure

```
├── 00-快速开始/        ← Quick Start
├── 01-收件箱/          ← Inbox (new input)
├── 02-对话记录/        ← Dialogue Records
├── 03-资产库/          ← Assets (reusable)
├── 04-输出成果/         ← Output
├── 05-知识沉淀/        ← Knowledge Base
├── 06-参考资料/        ← References
├── 07-项目文档/        ← Project Docs
├── 99-归档/            ← Archive
└── .workbuddy/       ← System Config
```

## Core Features

| Feature | Location |
|--------|----------|
| Knowledge Graph | wiki/ directory |
| Long-term Memory | .workbuddy/记忆层/ |
| AI Collaboration | .workbuddy/AI协作体系/ |
